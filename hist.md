### Python f-string 주의사항
- JS 리터럴 `{` `}` → `{{` `}}` 으로 이스케이프
- JS 템플릿 리터럴 `${var}` → `${{var}}` 으로 작성
- `${expr}` 에서 `expr` 에 추가 중괄호 없어야 함 (예: `${{risks.length-6}}` 이 맞음, `${{{risks.length-6}}}` 은 오류)
