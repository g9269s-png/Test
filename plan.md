### 대시보드 홈 — 현재 구성

```
┌─ KPI 카드 7개 (종사자이수율 / 취급자이수율 / 연간입고 / 연간출고 / 현재보관 / 검사리스크 / 전체리스크)
├─ row73 레이아웃
│   ├─ 좌 (7fr)
│   │   ├─ [교육이수현황] 종사자도넛 | 취급자도넛  ← 도넛 2개 병렬
│   │   └─ [월별 입출고 추이] 바 차트
│   └─ 우 (3fr)
│       └─ 리스크/일정 알림 카드 목록
├─ row2: 화학물질별 보관현황 (가스 가로 바) | 정기검사 현황 테이블
└─ AX 요약 텍스트
```

### 샘플 데이터 기준

- 기준일: 2026-06-13
- 종사자교육: 100명 (이수 84 / 미이수 16) — 9개 부서 배분
- 취급자교육: 21명 (기술기획파트 + 개발2실만, 이수 17 / 미이수 4)
- 화학물질: 17종 (가스 16 + KOH)
- 정기검사: 유해화학물질 9건 + 고압가스 7건 = 16건
- 가스감지기: 100슬롯 (샘플 10건, GD-003은 교정기한초과 상태)
- 일일점검 이력: 가스캐비닛 5건, N2탱크 5건

## 미완 / 개선 예정 항목

### 단기 (HTML 대시보드 내)

| 항목 | 내용 | 우선순위 |
|---|---|---|
| 홈 교육 섹션 통합 | 종사자+취급자 도넛을 한 박스로, 옆에 감지기 교정 요약 추가 | ★★★ |
| 가스감지기 100행 충원 | 실제 설치 감지기 정보 입력 필요 | ★★☆ |
| 일일점검 — 점검 항목 커스터마이징 | 현장 실제 항목과 맞추기 | ★★☆ |
| 교육 이수일 / 만료일 관리 | 차기 교육 예정일 표시 | ★☆☆ |
| 보관 현황 경보 기준 | 최대 보관 수량 대비 경보 | ★☆☆ |

### 중기 (기능 확장)

| 항목 | 내용 |
|---|---|
| 실제 데이터 입력 연결 | 현재는 샘플 고정값 → 엑셀 업로드 or 직접 입력 방식 연계 |
| 데이터 영속성 | 현재 페이지 새로고침 시 리셋 → LocalStorage 저장 검토 |
| 인쇄/PDF 출력 | 운영회의용 인쇄 레이아웃 |
| 화학물질 입출고 — 품목별 이력 필터 | 현재는 전체 이력만 |
| 리스크 알림 — 담당자 지정 + 조치 상태 관리 | 현재는 읽기 전용 |

### 장기 (시스템화)

| 항목 | 내용 |

| 사용자 인증 | 관리자/조회자 권한 분리 |
